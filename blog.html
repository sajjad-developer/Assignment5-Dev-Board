Question-1: What are the different ways to select an element in the DOM?
Two distinct methods to select an element in the DOM. For instance, document.getElementById() and document.querySelector()
Question-2: What is the difference between innerHTML, innerText, and textContent ?
InnerHTML: It includes text content of the element and html tags inside the parent container. Even comments are shown.
InnerText: It does not include html tags and displays visible text of the element inside the parent container.
TextContent: It also does not include html tags and displays visible and invisible text of the element inside the parent container.
Question-3: What is event delegation in the DOM?
Event is not handled where event is triggered. Event is handled under upper parent element.
Question-4: What is event bubbling in the DOM?
Event is propagated from child element to root element.
Question-5: How do you create, add, and remove elements using JavaScript?
Create element: document.createElement();
Add element: element.appendChild();
Remove element: element.removeChild();
